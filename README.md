## Sobre esse Projeto

[![Projetos em C][./images/logo.png]](https://github.com/RageHTML/C-exercicios-praticos)

Este repositório reúne alguns projetos básicos desenvolvidos em linguagem C, com foco na prática dos fundamentos da programação. Entre eles, estão o cálculo da área de uma circunferência e a simulação de um saldo final após um mês, aplicando operações matemáticas e lógica simples. Esses exemplos foram criados para reforçar conceitos essenciais e servir como base para estudos mais avançados.

## Primeiros Passos

1. Navegue até a pasta do projeto pelo terminal.
2. Compile o código:

   ```bash
   gcc main.c -o cal-area.c

### Pre Requesitos

- Um compilador C (gcc recomendado)
- Um terminal (Linux, macOS ou Windows com WSL/MinGW)

### Como baixar

Para obter este projeto, você pode clonar o repositório utilizando o Git:

```bash
git clone https://github.com/RageHTML/C-exercicios-praticos.git
```

## Uso do Projeto

Este projeto pode ser utilizado para:

- Praticar conceitos básicos da linguagem C
- Aprender operações matemáticas simples
- Entender entrada e saída de dados no terminal
